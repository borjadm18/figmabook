# figma-tokens

Converts Figma design tokens into `tailwind.config.ts` and `src/tokens/index.ts`.

**Trigger:** Invoked by figma-to-storybook orchestrator when phase=TOKENS.
**Input:** `.figma-migration.json` with manifest populated (including `manifest.variableDefs` if MCP mode).
**Output:** `tailwind.config.ts`, `src/tokens/index.ts`, `src/lib/utils.ts`.

---

## Data sources

- **MCP mode** (`config.mcp_mode: true`): Use variable definitions from `manifest.variableDefs` (collected during EXTRACT via `mcp__figma__get_variable_defs`). Variable names (e.g. `color/primary`, `spacing/md`) become token names. These are authoritative — supplement with manifest component values for anything not covered by Figma variables.
- **REST mode** (`config.mcp_mode: false`): Collect all unique values for `fills[].hex`, `fontFamily`, `fontSize`, `lineHeightPx`, and spacing values (`paddingTop`, `paddingRight`, `paddingBottom`, `paddingLeft`, `itemSpacing`) that appear in `config.tokenThreshold` or more components across the manifest.

## Token naming convention

| Figma variable / manifest field | `tailwind.config.ts` key | `src/tokens/index.ts` export |
|---|---|---|
| `color/primary` or most-used fill hex | `colors.primary` | `COLOR_PRIMARY` |
| `color/surface` | `colors.surface` | `COLOR_SURFACE` |
| `color/on-dark` | `colors['on-dark']` | `COLOR_ON_DARK` |
| `spacing/sm` or itemSpacing=8 (≥ tokenThreshold uses) | `spacing.sm` | `SPACING_SM` |
| `spacing/md` or itemSpacing=16 | `spacing.md` | `SPACING_MD` |
| `fontSize/body` or fontSize=16 + lineHeightPx=24 | `fontSize.body: ['16px', { lineHeight: '24px' }]` | `FONT_SIZE_BODY` |
| `borderRadius/base` or cornerRadius=3 | `borderRadius.base` | `BORDER_RADIUS_BASE` |
| `shadow/card` or DROP_SHADOW effect shared ≥ tokenThreshold | `boxShadow.card` | `BOX_SHADOW_CARD` |

## Arbitrary value rule

If a token value has no semantic Figma variable name AND appears in fewer than `config.tokenThreshold` components: do NOT add it to `tailwind.config.ts`. It will be used as an arbitrary Tailwind value inline in the component JSX (e.g. `px-[22px]`).

If a value IS shared (≥ tokenThreshold) but has no Figma variable name: generate a name from context. Examples: a padding value used only in CTA buttons → `spacing.cta-h`. A font size used in cards and organisms → `fontSize.display`.

## Output: tailwind.config.ts

Write to `tailwind.config.ts` in the working directory. If the file already exists, merge into `theme.extend` only — do not overwrite other config keys.

**Deep merge rule:** for each token category (`colors`, `spacing`, `fontSize`, etc.), append new keys. If a key already exists in the project config, keep the existing value and log a 🟡 Important notice:
> `Token conflict: tailwind.config.ts already defines {key} as {existingValue}. Figma value is {figmaValue}. Keeping existing value — update manually if needed.`

```ts
import type { Config } from 'tailwindcss';

const config: Config = {
  content: ['./src/**/*.{ts,tsx}'],
  theme: {
    extend: {
      colors: {
        // Auto-generated by figma-tokens — update token values in Figma, re-run figma-tokens
        primary:  '#FF6900',  // Figma: color/primary
        surface:  '#484853',  // Figma: color/surface
        'on-dark': '#FFFFFF', // Figma: color/on-dark
      },
      fontSize: {
        body: ['16px', { lineHeight: '24px' }],  // Figma: fontSize/body
      },
      spacing: {
        sm: '8px',   // Figma: spacing/sm
        md: '16px',  // Figma: spacing/md
      },
      borderRadius: {
        base: '3px',  // Figma: borderRadius/base
      },
      boxShadow: {
        card: '0 0 4px 0 rgba(0,0,0,0.25)',  // Figma: shadow/card
      },
    },
  },
  plugins: [],
};

export default config;
```

Every token entry must have a `// Figma: {variable name or source field}` comment.

## Output: src/tokens/index.ts

Create `src/tokens/index.ts`. Export a single `tokens` object with `as const` for full TypeScript narrowing, plus convenience type exports:

```ts
// Auto-generated by figma-tokens — do not edit manually
// Source: Figma file {config.figmaFileId}

export const tokens = {
  COLOR_PRIMARY:       '#FF6900',   // Figma: color/primary
  COLOR_SURFACE:       '#484853',   // Figma: color/surface
  COLOR_ON_DARK:       '#FFFFFF',   // Figma: color/on-dark
  FONT_SIZE_BODY:      '16px',      // Figma: fontSize/body
  LINE_HEIGHT_BODY:    '24px',      // Figma: fontSize/body lineHeight
  SPACING_SM:          '8px',       // Figma: spacing/sm
  SPACING_MD:          '16px',      // Figma: spacing/md
  BORDER_RADIUS_BASE:  '3px',       // Figma: borderRadius/base
  BOX_SHADOW_CARD:     '0 0 4px 0 rgba(0,0,0,0.25)', // Figma: shadow/card
} as const;

export type Tokens     = typeof tokens;
export type TokenKey   = keyof Tokens;
export type TokenValue = Tokens[TokenKey];
```

## Output: src/lib/utils.ts

Create `src/lib/utils.ts` if it does not exist. This file is required by the figma-component skill:

```ts
import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

Do not overwrite if the file already exists and already exports a `cn` function.
